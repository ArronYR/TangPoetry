# tp_frontend

A [React](http://reactjs.org) application for `Tang Poetry` Frontend.

See [demo](http://tpf.helloarron.com/)

![poetry](http://7xrgqs.com1.z0.glb.clouddn.com/poetry.gif)

## Install
```
git clone https://github.com/ArronYR/TangPoetry/
cd tp_frontend
npm install
npm start
```
