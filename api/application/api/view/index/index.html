响应参数 类型 说明

接口详情

1.1 根据id获取一篇诗

  • 请求URL

    /api/poetry/:id

  • 请求方式

    GET

  • 请求参数

    请求参数 参数类型 是否可选 默认值 参数说明
    id int 必填 0 诗词id
    lang string 可选 zh-cn 语言 zh-cn
    poet int 可选 1 是否包含诗人信息
    rows int 可选 1 内容按。？分割成数组

  • 返回示例

    {
    "result": {
        "poetry": {
            "id": "1314",
            "poet_id": "106",
            "content": "梧桐相待老，鸳鸯会双死。贞妇贵殉夫，舍生亦如此。波澜誓不起，妾心井中水。",
            "title": "琴曲歌辞。列女操",
            "poet": {
                "id": "106",
                "name": "孟郊"
            },
            "rows": [
                {
                    "idx": 0,
                    "content": "梧桐相待老，鸳鸯会双死。"
                }
            ]
        }
    },
    "code": 1001,
    "message": "成功获取诗信息"
}

                        {
                            "idx": 0,
                            "content": "重峦俯渭水，碧嶂插遥天。"
